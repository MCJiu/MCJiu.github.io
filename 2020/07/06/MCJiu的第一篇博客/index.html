博客大致介绍

  1. 博客的搭建和大致配置花费了 0 CNY !真是伟大的白嫖成就！
  2. 博客使用的域名是 blog.mcjiu.ml ，麻烦按Ctrl+D加入书签一波~ 至于域名以 .ml 结尾就不要吐槽了，因为这是免费域名，以 .com 结尾的域名需要付费，这样就不能达成第一条成就了。。
  3. 博客计划投放一些我未来的原创文章，例如原创书评和其他一些诡异的东西
  4. 博客的更新应该会非常缓慢，因为假期很少 （连暑假都只放5天假）
  5. 博客在更新的时候（包括更新配置和更新文章）会出现短暂的无法访问的情况，大致持续2-3min

博客配置

  1. 博客托管在Github，使用Cloudflare DNS服务，具有一定的反D-Dos性能。
  2. 博客使用TLS 1.3加密传输，但可能有部分引用使用http未加密协议。
  3. 由于博客使用了Cloudflare的TLS协议，若用Microsoft Edge访问可能出现TLS协议报错。请使用Chrome等浏览器访问。
  4. 博客所有者：MCJiu (或称 Vagrant)

OK,等着更新吧。。
